Basics Of Java Script

Arrays and Strings

MDN website for Lerning JS

In JavaScript, an Array is an ordered list of values. Each value is called an element, and each element has a numeric position in the array, known as its index. Arrays in JavaScript are zero-indexed, meaning the first element is at index 0, the second at index 1, and so on.

A JavaScript String is a sequence of characters, typically used to represent text. In JavaScript, there is no character type (Similar to Python and different from C, C++ and Java), so a single character string is used when we need a character. Like Java and Python, strings in JavaScript are immutable

Objects

An Object in JavaScript is a data structure used to store related data collections. It stores data as key-value pairs, where each key is a unique identifier for the associated value. Objects are dynamic, which means the properties can be added, modified, or deleted at runtime.

Optimation techniques

Optimization techniques are mathematical methods used to find the best solution to a problem, often by maximizing or minimizing a function, and are used across various fields like engineering, finance, and computer science.

Optimization techniques broadly encompass methods for finding the best solution to a problem, whether it's maximizing profit, minimizing cost, or finding the most efficient route. Some common techniques include:
linear programming, dynamic programming, and heuristic algorithms like genetic algorithms and simulated annealing.
